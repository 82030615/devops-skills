# Java 深度代码分析技能

## 简介

本技能帮助AI助手深度探索Java代码库中的特定功能，结合AST（抽象语法树）静态分析与LSP（语言服务协议）上下文，生成类图、状态机、时序图等Mermaid可视化图表，全面解构代码逻辑。

## 使用场景

- 🔍 理解复杂Java代码库的特定功能实现
- 📊 需要代码结构的可视化图表
- 🔗 追踪方法调用链和数据流向
- 🏗️ 分析状态机、设计模式
- ⚠️ 评估异常边界和并发安全性

## 触发方式

使用以下关键词触发技能：
- "分析Java功能"
- "Java代码深度分析"
- "代码逻辑解构"
- "Java AST分析"
- "LSP代码探索"

## 输出内容

技能将生成 `[功能名]_analysis.md` 文件，包含：

1. **静态结构分析**
   - 类图 (classDiagram)
   - 接口实现关系
   - 依赖关系图

2. **动态逻辑分析**
   - 状态机图 (stateDiagram-v2)
   - 时序图 (sequenceDiagram)
   - 控制流程图 (flowchart TD)

3. **边界与异常分析**
   - 未捕获异常路径
   - 边界参数识别
   - 并发安全性评估

## 工作原理

1. **拓扑扫描**：使用LSP识别核心类、接口、继承关系
2. **逻辑穿透**：追踪从入口到执行终点的完整调用链路
3. **模式识别**：识别设计模式和状态机
4. **可视化输出**：生成Mermaid图表

## 依赖要求

- 代码库需要配置Java LSP（如Eclipse JDT LS、IntelliJ IDEA）
- 或集成AST扫描工具（如Spoon、JavaParser）

## 注意事项

- 分析大型功能时，建议先关注核心路径
- 遇到循环依赖时会标注并截断
- 第三方库只分析Entry Point
- 不确定时会主动询问是否需要进一步跳转
